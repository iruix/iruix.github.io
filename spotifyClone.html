Spotify Clone

Project info

I started the project to practice writing PHP and get better. I never planned to work on the front end, but a friend promised to help me out. I did the project for couple of weeks, nearly completing the whole back end and currently I am working on Front End. End goal is to get all functions spotify has. Make playlists, Delete playlists, remove songs and many more. Database is created and can handle playlist data. I don't want to add algorithms, because I don't plan on using that many songs. I do not plan on publishing the site, it is only for personal development. I use HTML, CSS, JS, Jquery, PHP and bootstrap framework. I designed the front page using Bootstrap Studio.
